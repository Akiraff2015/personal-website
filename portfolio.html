Project 1

For the first project, we were assigned to build a game. For the technologies that I used to build my game were: HTML5 Canvas, JavaScript (jQuery) for DOM manipulation, CSS3. Key components of my game's algorithm: 1. Sprite sheet: due it's nature of sprite sheets being a fixed height and width, I designed an algorithm which crops specific positions, resulting in a unique image, from a larger image. The reason behind is, because I want to make sure that image gets loaded only one time which optimizes the loading time of my image. 2. Perlin Noise: I constructed a very basic Perlin Noise algorithm, in order to create randomness to my game. Thereby, every time when player refreshes the game, it will randomly design a new map for them.

Dynamically Generated Content: All contents were dynamically generated, without requiring the user to refresh the page. 2. Password Encryption: All user's password being stored in the database were encrypted with Salt and Hash.
